# Esanin - Bootstrap Website

A modern, responsive website built with Bootstrap 5 CSS framework.

## Features

✨ **Responsive Design** - Works seamlessly on all devices
🎨 **Beautiful UI** - Modern and clean interface
⚡ **Fast Performance** - Optimized for speed
🔧 **Easy to Customize** - Well-structured and documented code

## What's Included

- **index.html** - Main HTML file with Bootstrap components
- **style.css** - Custom CSS for enhanced styling
- **script.js** - JavaScript for interactive functionality

## Quick Start

1. Open `index.html` in your web browser
2. Customize the content as needed
3. Modify `style.css` for custom styling
4. Add more interactivity using `script.js`

## Bootstrap Components Used

- Navbar with responsive menu
- Hero section with call-to-action buttons
- Feature cards with hover effects
- Contact form
- Responsive grid system
- Typography and utilities

## Sections

1. **Navigation** - Responsive navbar with smooth scrolling
2. **Hero** - Welcome section with primary call-to-action
3. **Features** - Three feature cards showcasing key benefits
4. **About** - Information section with content and checklist
5. **Contact** - Contact form for user inquiries
6. **Footer** - Copyright and footer information

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #0d6efd;
    --secondary-color: #6c757d;
    --dark-color: #212529;
    --light-color: #f8f9fa;
}
```

### Content
Update text, images, and links directly in `index.html`

### Styling
Modify existing styles or add new ones in `style.css`
